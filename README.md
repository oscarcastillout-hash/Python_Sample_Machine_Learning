# Project Title

## Description
This project demonstrates a simple linear regression model using synthetic data. It covers data generation, model training, prediction, and visualization of results.

## Setup
To run this notebook, you will need the following libraries:
- `numpy`
- `matplotlib`
- `pandas`
- `scikit-learn`

You can install them using pip:
```bash
pip install numpy matplotlib pandas scikit-learn
```

## Usage
1. Run all cells in sequence to generate data, train the model, and visualize the results.
2. The notebook will display various plots and metrics related to the linear regression model.
